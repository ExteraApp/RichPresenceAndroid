# Extera RPC
Extera RPC (RPC as in Rich Presence) is a simple Android app for sending [MSC4320](https://github.com/matrix-org/matrix-spec-proposals/pull/4320) compatible rich presence information to Matrix homeserver.

This app uses **usage stats** for app detection (m.rpc.activity) and NotificationListenerService for music detection (m.rpc.media)
